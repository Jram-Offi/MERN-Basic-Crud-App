Front-End Project

Overview
This is the front-end of a web application built with React. It includes various dependencies for testing, routing, and general functionality. The project uses React Scripts for bundling and running the development server.

Dependencies
The project relies on the following dependencies:

-> @testing-library/jest-dom: Provides custom matchers for testing DOM elements.
-> @testing-library/react: A library for testing React components.
-> @testing-library/user-event: Simulates user interactions for testing.
-> axios: A promise-based HTTP client for making requests.
-> react: The React library for building user interfaces.
-> react-dom: The React DOM package for rendering React components.
-> react-router-dom: A package for handling routing in React applications.
-> react-scripts: A package that includes scripts for building, testing, and running React applications.
-> web-vitals: A library for measuring the core web vitals of your application.

Installation
To install the project and its dependencies, run:
-> npm install

Installing Commends:

-> npm install @testing-library/jest-dom@^5.17.0
-> npm install @testing-library/react@^13.4.0
-> npm install @testing-library/user-event@^13.5.0
-> npm install axios@^1.7.2
-> npm install react@^18.3.1
-> npm install react-dom@^18.3.1
-> npm install react-router-dom@^6.25.1
-> npm install react-scripts@5.0.1
-> npm install web-vitals@^2.1.4

Available Scripts
In the project directory, you can run the following commands:

npm start
Runs the app in development mode. Open http://localhost:3000 to view it in the browser.

npm run build
Builds the app for production to the build folder. It correctly bundles the app in production mode and optimizes the build for the best performance.

npm test
Launches the test runner in interactive watch mode.

npm run eject
Removes the react-scripts dependency and gives you full control over the configuration files.

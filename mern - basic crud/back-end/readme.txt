Back-End Project
Overview
This is the back-end of a web application built with Node.js, Express, and MongoDB. It uses Mongoose for MongoDB interaction and Cors for enabling cross-origin requests. Nodemon is used for automatic server restarts during development.

Dependencies
The project relies on the following dependencies:

-> cors: A package for enabling Cross-Origin Resource Sharing (CORS).
-> express: A web framework for Node.js to build the API.
-> mongoose: A MongoDB object modeling tool designed to work in an asynchronous environment.
-> nodemon: A tool that helps develop Node.js applications by automatically restarting the server when files change.

Installation
To install the project and its dependencies, run:
-> npm install

Installing commends:
-> npm install cors@^2.8.5
-> npm install express@^4.19.2
-> npm install mongoose@^8.5.2
-> npm install nodemon@^3.1.4
